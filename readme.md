#  Poker Time

A simple Python poker simulation project built with object-oriented programming.

This project creates playing cards, builds and shuffles a deck, deals 5-card poker hands, and checks for poker hand types such as flushes, straights, pairs, and straight flushes.

---

##  Project Overview

The project is split into two files:

- `Deck.py` — contains the `PlayingCard` and `Deck` classes
- `PokerHand.py` — contains the `PokerHand` class and a small simulation to estimate the probability of drawing a straight flush

---

## Features

- Create valid playing cards with suit and rank validation
- Represent a standard 52-card deck
- Shuffle the deck
- Deal cards from the deck
- Store poker hands safely
- Check for:
  - Flush
  - Straight
  - Straight flush
  - Pair
  - Two pair
  - Three of a kind
  - Full house
  - Four of a kind
- Run a simulation to estimate the probability of a straight flush
